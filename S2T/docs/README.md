## 项目概述

S2T是一个本地化语音转写系统，整合了三个原有项目的核心能力：

| 原项目 | 整合能力 |
|--------|----------|
| ProjectLocalASR | 批量文件转写、任务队列 |
| ProjectRec2Sum | 发言人分离、AI总结 |
| ProjectSound2Answer | 实时音频捕获、WebSocket |

## 核心特性

1. **多源音频**：文件、麦克风、系统声音
2. **智能转写**：FunASR + 发言人分离 + 热词
3. **AI能力**：总结、问答（流式输出）
4. **项目管理**：层级结构、编辑、搜索

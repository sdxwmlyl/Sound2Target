# 视频内容分析功能 — 需求文档

> 版本: v1.0 | 日期: 2026-07-11 | 状态: 待开发

---

## 1. 背景

用户需要从在线视频（Bilibili、YouTube 等）中提取完整内容并生成结构化总结。现有 S2T 系统只处理音频转写，缺少：
- 视频下载能力
- 视觉帧捕获与识别
- 音频+视觉内容的融合分析
- 基于多模态大模型的图表识别

## 2. 目标

### 2.1 核心目标
- 用户输入视频 URL → 系统自动完成：下载音频 → 转写 → 截图 → 多模态识别 → 融合总结
- 支持用户自定义总结要求（如"提取评测维度"、"列出所有数据表格"）

### 2.2 非目标
- 不做实时视频流处理
- 不做视频下载后的本地存储（仅保留音频和截图）
- 不替代 ComfyUI/视频编辑等专业工具

## 4. 功能需求

### FR-1：视频音频提取
- **输入**: 视频 URL（支持 Bilibili、YouTube 等主流平台）
- **处理**: 使用 yt-dlp 下载音频，转为 WAV 格式
- **输出**: 音频文件路径
- **约束**: 仅下载音频，不下载视频；超时 5 分钟

### FR-2：音频转写
- **输入**: WAV 音频文件
- **处理**: 复用现有 S2T ASR 引擎（FunASR）
- **输出**: 带时间戳的转写文案（segments）
- **约束**: 复用现有转写流程，无额外开发

### FR-3：视觉帧捕获
- **输入**: 视频 URL + 采样间隔（默认 30 秒）
- **处理**: 使用 Playwright/浏览器打开视频 → 逐帧截图
- **输出**: 截图文件列表（带时间戳）
- **约束**: 需要浏览器环境；Bilibili 需要登录态

### FR-4：文案关键词筛选
- **输入**: 转写文案（segments）
- **处理**: 关键词匹配 + 分级（Tier 1-4）
- **输出**: 关键帧时间点列表
- **约束**: 关键词可配置

### FR-5：多模态图表识别
- **输入**: 截图文件列表
- **处理**: 调用多模态大模型 API（如 Qwen-VL、GPT-4o）识别图表内容
- **输出**: 每帧的结构化描述（图表类型、数据、标题）
- **约束**: 需要配置多模态大模型 API

### FR-6：双轨融合分析
- **输入**: 转写文案 + 视觉帧描述
- **处理**: 按时间对齐，合并去重，生成完整内容清单
- **输出**: 带时间戳的完整内容清单（JSON）
- **约束**: 时间对齐窗口 15 秒

### FR-7：自定义总结
- **输入**: 完整内容清单 + 用户自定义要求
- **处理**: 调用 LLM（文本模型即可）按用户要求生成总结
- **输出**: Markdown 格式的结构化总结
- **约束**: LLM 使用现有配置（ollama/aliyun/deepseek）

### FR-8：系统配置
- **新增配置项**:
  ```yaml
  video:
    yt_dlp_path: "yt-dlp"  # yt-dlp 可执行文件路径
    sample_interval: 30     # 默认采样间隔（秒）
    max_duration: 3600      # 最大视频时长（秒）
    browser_timeout: 300    # 浏览器超时（秒）
  
  multimodal:
    provider: "openai"      # openai / aliyun / dashscope
    base_url: "https://dashscope.aliyuncs.com/compatible-mode/v1"
    model: "qwen-vl-max"
    api_key: ""
  ```

## 5. 非功能需求

| 需求 | 指标 |
|------|------|
| 性能 | 17 分钟视频全流程 < 10 分钟 |
| 可靠性 | 单步失败不影响已完成步骤的产出 |
| 可扩展性 | 新增视频平台只需适配 yt-dlp |
| 兼容性 | 复用现有 S2T 数据库和 API 结构 |

## 6. 依赖

| 依赖 | 版本 | 用途 |
|------|------|------|
| yt-dlp | 2026.7+ | 视频音频提取 |
| Playwright | 已有 | 视觉帧捕获 |
| FunASR | 已有 | 音频转写 |
| 多模态大模型 API | - | 图表识别（需配置） |
| 文本 LLM | 已有 | 自定义总结 |
